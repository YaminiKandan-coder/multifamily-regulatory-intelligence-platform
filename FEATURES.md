# Features

## Chat Q&A Agent
- Jurisdiction-specific filtering via cascading dropdowns
- Conversation history
- Sources displayed in collapsible sections

## Lease Compliance Checker
- PDF/DOCX upload
- Clause-by-clause assessment
- Violations, applicable regulations, suggested revisions
- Overall compliance score

## Regulation Explorer
- Semantic search
- Filter by category and location
- Source links

## Update Log
- Change detection across jurisdictions
- Deduplication by source URL
- Expandable entries

## Email Alerts
- Subscribe for immediate notifications
- Daily digests

## Rule-based Fallback
- Detects ESA fee violations
- Security deposit timeline
- Unreasonable late fees
- Rent increase caps

## Settings Page
- CSV loading
- Vector index initialization
- API key configuration
- Scraper trigger

## Source Registry
- Full CRUD on scrape-target sources
- CSV import/export
- DB vs CSV toggle
- URL testing

## Coverage
- Pet regulations (ESA, breed restrictions)
- Rental insurance requirements
- All 50 US states
